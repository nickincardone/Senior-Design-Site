## Dependencies

### Node.js

[Install Here](http://nodejs.org/download/)

Ensure node is installed

`npm`

### Grunt.js

run `npm install -g grunt-cli` 

### Bower

run `npm install -g bower`

## Setup

Clone repo and run:
`npm install && bower install`

Now run `grunt` and open `index.html`
