## Dependencies

[Install Here](http://nodejs.org/download/)

Ensure node is installed

`npm`

### Grunt.js

run `npm install -g grunt-cli` 

## Setup

Clone repo and run:
`sudo npm install` which will install all node dependicies

Now run `grunt && node server` and navigate to port 8080
